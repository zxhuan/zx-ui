# zx-ui

> Some mobile terminal UI components based on Vue include forms, lists, shells, buttons, tabs, etc

## component

``` bash
# button
# input
# cell
# dialog
# form 
# tab
# select
# radio
# checkbox
# popup

```
## Install
``` bash
npm install zx-ui -S
```
## Quick Start
``` bash
import Vue from 'vue'
import Zxui from 'zx-ui'
import 'zx-ui/lib/zxhuan.css' 
Vue.use(Zxui)
```
## Look at Examples
```
git clone https://github.com/zxhuan/zx-ui.git
cd zx-ui
npm install
npm run start
```
