# zx-ui

> 基于vue 写的一些移动端UI组件，包含表单，列表，弹层，按钮，选项卡等

## component

``` bash
# button
# input
# cell
# dialog
# form 
# tab
# select
# radio
# checkbox
# popup

```
## Install
``` bash
npm install zx-ui -S
```
## Quick Start
``` bash
import Vue from 'vue'
import Zxui from 'zx-ui'
import 'zx-ui/lib/zxhuan.css' 
Vue.use(Zxui)
```
## Look at Examples
```
git clone https://github.com/zxhuan/zx-ui.git
cd zx-ui
npm install
npm run start
```
